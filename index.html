fsharp-tapl

Code and Examples from Benjamin Pierce's "Types and Programming Languages", a comprehensive introduction to type systems and programming language theory. The code which accompanies the book is written in OCaml. This repository contains an F# port of that code, using FsLexYacc for lexing and parsing.

The fsharp-tapl library can be installed from NuGet: 
PM> Install-Package fsharp-tapl

Getting Started

Run the build script to download dependencies, build all projects, and run regression tests.

All projects are console apps. Run with file input

``

cd bin\relase untyped -i ....\untyped\test.f or input from command line arith -s "iszero (pred (succ 0));" ``
